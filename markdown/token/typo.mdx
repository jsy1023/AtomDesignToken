# Typography

타이포그래피는 문자와 글자를 시각적으로 배열하는 기술이자 예술입니다. 글꼴(font), 크기(size), 자간(letter-spacing), 행간(line-height), 정렬(alignment) 등 텍스트의 형태와 구조를 조정하여 가독성과 심미성을 높이는 역할을 합니다.
