# 네비게이션 (Navigation)

웹사이트나 애플리케이션의 구조를 탐색하고 주요 영역으로 이동할 수 있게 해주는 핵심 구성 요소입니다.

<Callout type="note">
  Atomsystem의 네비게이션은 상단(Top) 및 측면(Side) 레이아웃을 모두 지원하며, 모듈형 구조로 설계되어 자유로운 커스터마이징이 가능합니다.
</Callout>

---

## 상단 네비게이션 (Top Navigation)

페이지 상단에 고정되거나 배치되는 일반적인 네비게이션 형태입니다.

<ComponentPreview name="Navigation">
  <NavBrand href="#">Atomsystem</NavBrand>
  <NavList>
    <NavItem href="#">소개</NavItem>
    <NavItem href="#">컴포넌트</NavItem>
    <NavItem href="#">가이드</NavItem>
  </NavList>
</ComponentPreview>

---

## 사이드 네비게이션 (Side Navigation)

좌측 또는 우측에 배치되어 상세 메뉴 구조를 보여주는 네비게이션입니다. `Sidemenu` 컴포넌트를 함께 사용합니다.

<ComponentPreview name="Sidemenu">
  <NavList>
    <p className="px-4 py-2 text-sm font-bold opacity-50">카테고리</p>
    <NavItem href="#">데시보드</NavItem>
    <NavItem href="#">사용자 관리</NavItem>
    <NavItem href="#">설정</NavItem>
  </NavList>
</ComponentPreview>

## 설치 및 구성 (Installation)

<Steps>

### 컴포넌트 추가
터미널에서 필요한 컴포넌트들을 추가합니다.

```bash
npx atomsystem-add Navigation
npx atomsystem-add Sidemenu
```

### 소스 코드 확인
생성된 `Navigation.tsx`와 `Sidemenu.tsx`의 내부 구현을 확인해 보세요.

</Steps>

## 사용 가이드 (Usage)

### 기본 구조
네비게이션은 `Navigation`, `NavBrand`, `NavList`, `NavItem`의 조합으로 구성됩니다.

```tsx
<Navigation>
  <NavBrand href="/">Brand</NavBrand>
  <NavList>
    <NavItem href="/home">Home</NavItem>
  </NavList>
</Navigation>
```
