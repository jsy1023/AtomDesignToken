# 네비게이션

웹사이트, 어플리케이션의 전체 구조에서 주요 영역으로 이동할 수 있도록 하는 네비게이션 메뉴입니다. 상단 네비게이션과 사이드 네비게이션으로 구분하여 사용할 수 있습니다.

---

## Example

### Top Navigation

상단에 위치하는 네비게이션 메뉴입니다. `type="topnav"` 속성을 사용합니다.
