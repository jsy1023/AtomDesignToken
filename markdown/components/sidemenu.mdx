# 사이드메뉴 (Sidemenu)

좌측 또는 우측에 배치되어 상세 메뉴 구조를 보여주는 사이드 네비게이션 컴포넌트입니다.

<Callout type="note">
  `Sidemenu`, `NavList`, `NavItem`의 모듈형 구조로 구성됩니다. 현재 경로에 해당하는 `NavItem`은 자동으로 활성화 스타일이 적용됩니다.
</Callout>

---

## 사용 가이드 (Usage)

### 기본 구조
`Sidemenu`와 `NavList`, `NavItem`의 조합으로 구성됩니다.

```tsx
<Sidemenu minWidth={240}>
  <NavList>
    <p className="px-4 py-2 text-sm font-bold opacity-50">카테고리</p>
    <NavItem href="/dashboard">대시보드</NavItem>
    <NavItem href="/users">사용자 관리</NavItem>
    <NavItem href="/settings">설정</NavItem>
  </NavList>
</Sidemenu>
```

### 레이아웃 구성 (Next.js)

일반적으로 사이드메뉴는 메인 콘텐츠 영역과 Flex 또는 Grid 형태로 결합됩니다. Next.js의 `layout.tsx`에 배치하여 중첩 라우팅에서의 공통 UI로 활용할 수 있습니다.

```tsx
import { Sidemenu, NavList, NavItem } from "@/app/components/Sidemenu";

export default function DashboardLayout({ children }: { children: React.ReactNode }) {
  return (
    <div className="flex h-screen w-full overflow-hidden">
      {/* 1. 사이드메뉴 (좌측 고정) */}
      <Sidemenu minWidth={250}>
        <NavList>
          <p className="px-4 py-2 text-sm opacity-50">Menu</p>
          <NavItem href="/dashboard">Overview</NavItem>
          <NavItem href="/dashboard/analytics">Analytics</NavItem>
        </NavList>
      </Sidemenu>

      {/* 2. 메인 콘텐츠 영역 */}
      <main className="flex-1 overflow-auto bg-[var(--bg-document)]">
        {children}
      </main>
    </div>
  );
}
```

## 속성 (Properties)

### Sidemenu

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `children` | `React.ReactNode` | — | 내부에 배치할 콘텐츠 (`NavList` 등) |
| `minWidth` | `number` | — | 사이드메뉴의 최소 너비 (px 단위) |

### NavItem

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `href` | `string` | — | 이동할 경로. 현재 경로와 일치하면 활성화 스타일 적용 |
| `children` | `React.ReactNode` | — | 메뉴 텍스트 |
