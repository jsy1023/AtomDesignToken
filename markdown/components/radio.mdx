# 라디오 (Radio)

여러 선택지 중 단 하나만 선택할 수 있는 배타적 선택 컨트롤 요소입니다.

<Callout type="tip">
  사용자가 선택지를 한 눈에 비교하고 하나만 결정해야 할 때 사용합니다. 여러 개를 중복 선택해야 하는 경우에는 체크박스(Checkbox)를 사용하세요.
</Callout>

---

## 기본 사용법 (Basics)

라디오 버튼은 같은 `name` 속성을 공유하여 하나의 그룹으로 동작합니다.

<ComponentPreview name="Radio">
  <div className="flex gap-4">
    <Radio name="fruit" value="사과" defaultChecked />
    <Radio name="fruit" value="오렌지" />
    <Radio name="fruit" value="포도" />
  </div>
</ComponentPreview>

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 폼(Form) 컴포넌트 시스템을 추가합니다.

```bash
npx atomsystem-add Form
```

### 소스 코드 확인
`app/templates/Form/Form.tsx` 파일 내에서 `Radio` 구현 코드를 확인할 수 있습니다.

<ComponentSource name="Radio" />

</Steps>

## 다양한 예시 (Examples)

### 라벨과 그룹화 (With Group Label)

여러 라디오 버튼을 그룹으로 묶어 설명을 추가할 수 있습니다.

<ComponentPreview name="Radio">
  <div className="flex flex-col space-y-2">
    <Label label="배송 방법 선택" required />
    <Radio name="shipping" value="일반 배송" defaultChecked />
    <Radio name="shipping" value="새벽 배송" />
    <Radio name="shipping" value="당일 수령" />
  </div>
</ComponentPreview>

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `name` | `string` | `-` | 라디오 그룹을 식별하는 고유 이름 |
| `value` | `string` | `-` | 라디오 버튼의 값 |
| `label` | `string` | `-` | (옵션) 상단에 표시될 설명 텍스트 |
| `defaultChecked` | `boolean` | `false` | 초기 선택 상태 여부 |
| `disabled` | `boolean` | `false` | 비활성화 여부 |
