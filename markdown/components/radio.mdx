# 라디오 (Radio)

여러 선택지 중 단 하나만 선택할 수 있는 배타적 선택 컨트롤 요소입니다.

<Callout type="tip">
  사용자가 선택지를 한 눈에 비교하고 하나만 결정해야 할 때 사용합니다. 여러 개를 중복 선택해야 하는 경우에는 체크박스(Checkbox)를 사용하세요.
</Callout>

---

## 기본 사용법 (Basics)

라디오 버튼은 같은 `name` 속성을 공유하여 하나의 그룹으로 동작합니다.

---

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `name` | `string` | `-` | 라디오 그룹을 식별하는 고유 이름 |
| `value` | `string` | `-` | 라디오 버튼의 값 |
| `label` | `string` | `-` | (옵션) 상단에 표시될 설명 텍스트 |
| `defaultChecked` | `boolean` | `false` | 초기 선택 상태 여부 |
| `disabled` | `boolean` | `false` | 비활성화 여부 |
