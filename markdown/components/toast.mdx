# 토스트 알림 (Toast)

사용자에게 작업의 결과나 시스템의 상태를 짧고 명확하게 전달하는 피드백 요소입니다.

<Callout type="note">
  일정 시간이 지나면 자동으로 사라지며, 사용자의 현재 작업을 방해하지 않으면서 중요한 정보를 전달하는 데 사용됩니다.
</Callout>

---

## 활용 가이드 (Usage)

### 상태별 호출
`status` 속성을 사용하여 알림의 성격을 구분할 수 있습니다.

```tsx
const toast = useToast();

// 성공 알림
toast({
  status: "success",
  message: "저장되었습니다.",
});

// 에러 알림
toast({
  status: "danger",
  message: "오류가 발생했습니다.",
  desc: "잠시 후 다시 시도해 주세요."
});
```
