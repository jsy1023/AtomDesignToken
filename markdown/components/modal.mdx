# 모달 (Modal)

사용자의 작업 흐름을 일시 정지시키고, 중요한 정보 전달이나 특정 작업 수행을 요구하는 오버레이 창입니다.

<Callout type="warning" title="주의 사항">
  모달은 사용자의 흐름을 강제로 중단시키므로, 꼭 필요한 경우에만 절제하여 사용하는 것이 좋습니다.
</Callout>

## 설치 (Installation)

<Steps>

### 컴포넌트 추가
터미널에서 아래 명령어를 실행합니다.

```bash
npx atomsystem-add Modal
```

### 상위 레이아웃 설정
모달이 올바르게 렌더링되도록 `layout.tsx` 파일 최상위에 `Modal` 컨테이너를 추가해야 합니다.

```tsx
import { Modal } from "@/app/templates/Modal/Modal";

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Modal /> {/* 모달 포털 포인트 */}
      </body>
    </html>
  );
}
```

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `isOpen` | `boolean` | `false` | 모달의 활성화 상태 |
| `onRequestClose` | `() => void` | `-` | 닫기 요청 시 호출되는 핸들러 |
| `modalTitle` | `string` | `-` | 상단에 표시될 제목 |
| `modalFooter` | `ReactNode` | `-` | 하단에 배치할 컴포넌트(주로 버튼) |
| `size` | `normal` \| `large` \| `full` | `normal` | 모달의 너비 크기 |
