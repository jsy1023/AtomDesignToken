# 목차 (TOC - Table of Contents)

페이지의 전체적인 구조를 한 눈에 파악하고 원하는 섹션으로 빠르게 이동할 수 있게 돕는 내비게이션 요소입니다.

<Callout type="tip">
  내용이 긴 문서 페이지에서 사용자의 탐색 편의성을 높이기 위해 우측 사이드바 등에 배치하는 것을 권장합니다.
</Callout>

---

## 기본 사용법 (Basics)

현재 페이지의 제목(`h2`, `h3` 등)을 수집하여 리스트로 보여줍니다.

## 활용 가이드 (Usage)

### 자동 스크롤 추적
TOC 컴포넌트는 사용자의 스크롤 위치를 추적하여 현재 읽고 있는 섹션을 시각적으로 강조해 줍니다. 별도의 설정 없이 페이지 내의 제목 태그들을 자동으로 인식합니다.

```tsx
import TOC from "@/app/templates/TOC/TOC";

export default function Page() {
  return (
    <div className="flex">
      <main>...</main>
      <aside>
        <TOC />
      </aside>
    </div>
  );
}
```
