# 탭 (Tab)

내용을 카테고리별로 분류하여 공간을 효율적으로 전환하며 보여주는 UI 요소입니다.

<Callout type="note">
  사용자가 현재 맥락을 유지하면서 관련 있는 다른 콘텐츠로 쉽고 빠르게 이동할 수 있게 돕습니다.
</Callout>

---

## 기본 사용법 (Basics)

탭 메뉴와 각 탭에 대응하는 콘텐츠 리스트를 정의합니다.

<ComponentPreview name="Tab">
  <Tab
    tabs={[
      { id: "html", target: ["html"], tabItem: "HTML" },
      { id: "css", target: ["css"], tabItem: "CSS" },
      { id: "js", target: ["js"], tabItem: "JavaScript" },
    ]}
    tabContents={[
      { id: "c1", tag: "html", content: <p>HTML은 웹 페이지의 구조를 정의합니다.</p> },
      { id: "c2", tag: "css", content: <p>CSS는 웹 페이지의 스타일과 레이아웃을 정의합니다.</p> },
      { id: "c3", tag: "js", content: <p>JS는 웹 페이지에 상호작용과 동적인 기능을 부여합니다.</p> },
    ]}
  />
</ComponentPreview>

## 활용 가이드 (Usage)

### 데이터 구조
탭은 `tabs` (메뉴 설정)와 `tabContents` (매칭될 내용) 두 가지 배열 데이터가 필요합니다. `target`과 `tag` 값이 일치하는 항목끼리 연결됩니다.

```tsx
<Tab
  tabs={[
    { id: "tab1", target: ["groupA"], tabItem: "메뉴1" }
  ]}
  tabContents={[
    { id: "content1", tag: "groupA", content: <Box /> }
  ]}
/>
```

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `tabs` | `Array` | `[]` | 탭 버튼 설정 (id, target[], tabItem) |
| `tabContents` | `Array` | `[]` | 탭 내용 설정 (id, tag, content) |
