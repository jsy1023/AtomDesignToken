# 탭 (Tab)

내용을 카테고리별로 분류하여 공간을 효율적으로 전환하며 보여주는 UI 요소입니다.

<Callout type="note">
  사용자가 현재 맥락을 유지하면서 관련 있는 다른 콘텐츠로 쉽고 빠르게 이동할 수 있게 돕습니다.
</Callout>

## 활용 가이드 (Usage)

### 개별 컴포넌트 조합 구조 (Compound Components)
탭은 `<Tabs>`, `<TabsList>`, `<TabsTrigger>`, `<TabsContent>`를 조합하여 구성합니다. `defaultValue`와 `value` 값이 일치하는 항목끼리 연결됩니다.

```tsx
<Tabs defaultValue="tab1">
  <TabsList>
    <TabsTrigger value="tab1">메뉴 1</TabsTrigger>
  </TabsList>
  <TabsContent value="tab1">
    <Box />
  </TabsContent>
</Tabs>
```
