# 배지 (Badge)

상태, 수량, 라벨 등 짧은 텍스트 정보를 시각적으로 강조하여 전달하는 컴포넌트입니다.

<Callout type="tip">
  배지는 읽기 전용 요소입니다. 클릭 가능한 액션이 필요한 경우 버튼 컴포넌트 사용을 고려하세요.
</Callout>
