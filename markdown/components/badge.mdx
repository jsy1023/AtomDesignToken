# 배지 (Badge)

상태, 수량, 라벨 등 짧은 텍스트 정보를 시각적으로 강조하여 전달하는 컴포넌트입니다.

<Callout type="tip">
  배지는 읽기 전용 요소입니다. 클릭 가능한 액션이 필요한 경우 버튼 컴포넌트 사용을 고려하세요.
</Callout>

---

## 기본 사용법 (Basics)

다양한 색상의 배지를 생성할 수 있습니다. `color` 속성을 사용하여 의미를 부여합니다.

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 아래 명령어를 실행합니다.

```bash
npx atomsystem-add Badge
```

## 스타일 변형 (Variants)

### 틴트 타입 (Tinted)

`type="tint"` 속성을 사용하면 배경색을 은은하게 강조할 수 있습니다.

<ComponentPreview name="Badge" color="primary" type="tint">
  Tinted Primary
</ComponentPreview>

<ComponentPreview name="Badge" color="success" type="tint">
  Tinted Success
</ComponentPreview>

---

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `color` | `primary` \| `secondary` \| `success` \| `warning` \| `danger` | `primary` | 배지의 색상 스타일 |
| `type` | `solid` \| `tint` | `solid` | 배지의 채우기 스타일 |
