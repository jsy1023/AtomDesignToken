# 콜랩스 (Collapse)

많은 양의 콘텐츠를 제목 아래 숨겨두었다가 클릭 시 펼쳐서 보여주는 아코디언 스타일의 컴포넌트입니다.

<Callout type="note">
  정보의 위계를 정리하고 화면 공간을 효율적으로 사용하고 싶을 때 유용합니다.
</Callout>

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 아래 명령어를 실행합니다.

```bash
npx atomsystem-add Collapse
```

### 소스 코드 확인
`app/templates/Collapse/Collapse.tsx`에서 애니메이션과 레이아웃 로직을 확인하세요.

</Steps>

## 스타일 변형 (Variants)

### 아코디언 모드 (Accordion)

`type="accordion"` 속성을 사용하면 한 번에 하나의 항목만 펼쳐지도록 제한할 수 있습니다.
