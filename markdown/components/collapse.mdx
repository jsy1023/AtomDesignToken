# 콜랩스 (Collapse)

많은 양의 콘텐츠를 제목 아래 숨겨두었다가 클릭 시 펼쳐서 보여주는 아코디언 스타일의 컴포넌트입니다.

<Callout type="note">
  정보의 위계를 정리하고 화면 공간을 효율적으로 사용하고 싶을 때 유용합니다.
</Callout>

---

## 기본 사용법 (Basics)

`CollapseProvider`, `Collapse`, `CollapseItem`, `CollapseHeader`, `CollapseContent`를 조합하여 자유롭게 구성할 수 있는 모듈형 구조입니다.

<ComponentPreview name="collapse-demo" />

```tsx
import { 
  Collapse, 
  CollapseProvider, 
  CollapseItem, 
  CollapseHeader, 
  CollapseContent 
} from "@/app/templates/Collapse/Collapse";

export default function Demo() {
  return (
    <CollapseProvider>
      <Collapse>
        <CollapseItem value="1">
          <CollapseHeader>첫 번째 항목</CollapseHeader>
          <CollapseContent>콘텐츠 내용입니다.</CollapseContent>
        </CollapseItem>
      </Collapse>
    </CollapseProvider>
  );
}
```

## 스타일 변형 (Variants)

### 아코디언 모드 (Accordion)

`CollapseProvider`에 `type="accordion"` 속성을 사용하면 한 번에 하나의 항목만 펼쳐지도록 제한할 수 있습니다.

## 속성 (Properties)

### CollapseProvider
| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| type | `"accordion" \| "collapse"` | `"collapse"` | 펼침 동작 모드 |
| defaultActiveKeys | `(string \| number)[]` | `[]` | 초기 활성화된 항목들의 키 |

### CollapseItem
| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| value | `string \| number` | - | 항목의 고유 식별자 (필수) |
| className | `string` | - | 추가 스타일 클래스 |
