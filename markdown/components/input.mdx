# 입력창 (Input)

사용자로부터 데이터를 입력받기 위한 기본적인 대화형 컨트롤 요소입니다.

<Callout type="note">
  입력창은 포커스, 에러, 비활성 등 다양한 상태를 직관적으로 표현하여 사용자와 원활하게 상호작용해야 합니다.
</Callout>

---

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 폼(Form) 컴포넌트 시스템을 추가합니다.

```bash
npx atomsystem-add Form
```

### 소스 코드 확인
`app/templates/Form/Form.tsx` 파일 내에서 `Input` 구현 코드를 확인할 수 있습니다.

<ComponentSource name="Input" />

</Steps>

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `type` | `string` | `text` | 입력 타입 (text, password, email 등) |
| `placeholder` | `string` | `-` | 입력 도움말 텍스트 |
| `disabled` | `boolean` | `false` | 비활성화 여부 |
| `required` | `boolean` | `false` | 필수 입력 여부 |
| `readOnly` | `boolean` | `false` | 읽기 전용 여부 |
