# 체크박스 (Checkbox)

여러 가지 선택 사항 중 하나 이상을 선택할 수 있게 해주는 컨트롤 요소입니다.

<Callout type="tip">
  선택지가 1개에서 10개 사이일 때 사용을 권장하며, 단일 선택만 필요한 경우에는 라디오(Radio) 버튼 사용을 고려하세요.
</Callout>

---

## 기본 사용법 (Basics)

가장 기본적인 형태의 체크박스입니다.

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 폼(Form) 컴포넌트 시스템을 추가합니다.

```bash
npx atomsystem-add Form
```

### 소스 코드 확인
`app/templates/Form/Form.tsx` 파일 내에서 `Checkbox` 구현 코드를 확인할 수 있습니다.

<ComponentSource name="Checkbox" />

</Steps>

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `text` | `string` | `-` | 체크박스 옆에 표시될 라벨 텍스트 |
| `disabled` | `boolean` | `false` | 체크박스 비활성화 여부 |
| `defaultChecked` | `boolean` | `false` | 초기 선택 상태 여부 |
| `required` | `boolean` | `false` | 필수 선택 항목 여부 |
