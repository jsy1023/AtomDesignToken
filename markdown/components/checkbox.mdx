# 체크박스 (Checkbox)

여러 가지 선택 사항 중 하나 이상을 선택할 수 있게 해주는 컨트롤 요소입니다.

<Callout type="tip">
  선택지가 1개에서 10개 사이일 때 사용을 권장하며, 단일 선택만 필요한 경우에는 라디오(Radio) 버튼 사용을 고려하세요.
</Callout>
