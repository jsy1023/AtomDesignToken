# 네비게이션 (Navigation)

웹사이트나 애플리케이션의 구조를 탐색하고 주요 영역으로 이동할 수 있게 해주는 상단(Top) 네비게이션 컴포넌트입니다.

<Callout type="note">
  `Navigation`, `NavBrand`, `NavList`, `NavItem`의 모듈형 구조로 설계되어 자유로운 커스터마이징이 가능합니다. 현재 경로에 해당하는 `NavItem`은 자동으로 활성화 스타일이 적용됩니다.
</Callout>
