# 테이블 (Table)

복잡한 데이터를 행(row)과 열(column)의 격자 구조로 시각화하여 구조적으로 보여주는 요소입니다.

<Callout type="note">
  많은 양의 정보를 비교하거나 목록 형태로 정리하여 보여줄 때 가장 효율적인 도구입니다.
</Callout>

---

## 기본 사용법 (Basics)

객체 배열 형태의 데이터를 `tableData` 속성으로 전달하면 자동으로 헤더와 행이 생성됩니다.

<ComponentPreview name="Table">
  <Table
    tableData={[
      { "아이디": "user01", "이름": "홍길동", "상태": "활성" },
      { "아이디": "user02", "이름": "김철수", "상태": "대기" },
      { "아이디": "user03", "이름": "이영희", "상태": "활성" },
    ]}
  />
</ComponentPreview>

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 아래 명령어를 실행합니다.

```bash
npx atomsystem-add Table
```

## 다양한 예시 (Examples)

### 데이터가 없을 때 (Empty State)

데이터가 비어있는 경우 정해진 안내 문구를 표시합니다.

<ComponentPreview name="Table">
  <Table tableData={[]} />
</ComponentPreview>

---

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `tableData` | `Record<string, any>[]` | `[]` | 테이블에 표시할 데이터 배열. 객체의 키가 헤더가 됩니다. |
