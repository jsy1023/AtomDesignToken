# 카드 (Card)

정보를 논리적으로 그룹화하고 시각적으로 구분하여 보여주는 컨테이너 요소입니다.

<Callout type="note">
  카드는 헤더, 콘텐츠, 푸터로 구성된 모듈형 구조를 가지고 있어 유연하게 배치할 수 있습니다.
</Callout>

## 설치 (Installation)

<Steps>

### 컴포넌트 추가
터미널에서 아래 명령어를 실행하여 카드 시스템을 추가합니다.

```bash
npx atomsystem-add Card
```

## 활용 가이드 (Usage)

### 복합 구성
카드는 내부에 버튼, 이미지 등 다른 컴포넌트를 포함하여 풍부한 정보를 담을 수 있습니다.

```tsx
<Card className="w-full">
  <CardHeader className="font-bold">제품명</CardHeader>
  <CardContent>제품에 대한 상세 설명이 들어가는 영역입니다.</CardContent>
  <CardFooter className="flex justify-end">
    <Button size="sm">상세보기</Button>
  </CardFooter>
</Card>
```

---

## 구성 요소 (Components)

- **`Card`**: 메인 컨테이너
- **`CardHeader`**: 제목 영역
- **`CardContent`**: 본문 영역
- **`CardFooter`**: 하단 영역
