# 사이드메뉴 (Sidebar)

좌측 또는 우측에 배치되어 상세 메뉴 구조를 보여주는 사이드 네비게이션 컴포넌트입니다.

<Callout type="note">
  `SidebarProvider`, `Sidebar`, `SidebarToggle`, `SideNavList`, `SideNavLink`의 모듈형 구조로 설계되어 자유로운 커스터마이징이 가능합니다. GSAP을 활용한 부드러운 슬라이딩 애니메이션과 상태 관리 기능이 포함되어 있습니다.
</Callout>

---

## 기본 사용법 (Basics)

카테고리 레이블과 메뉴 항목을 조합하여 계층형 사이드 메뉴를 구성합니다. 상단 헤더나 네비게이션바에 토글 버튼을 배치하여 열고 닫을 수 있습니다.

### 레이아웃 구성 (Next.js)

일반적으로 사이드메뉴는 메인 콘텐츠 영역과 Flex 형태로 결합됩니다. Next.js의 `layout.tsx`에 `SidebarProvider`를 배치하여 공통 UI로 활용할 수 있습니다.

```tsx
import { Sidebar, SidebarProvider, SidebarToggle, SideNavList, SideNavLink, SideNavItem } from "@/app/templates/Sidebar/Sidebar";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <SidebarProvider>
      <div className="flex h-screen w-full overflow-hidden">
        {/* 1. 사이드메뉴 (좌측 고정) */}
        <Sidebar minWidth={250}>
          <SideNavList>
            <p className="px-4 py-2 text-sm opacity-50 uppercase">Menu</p>
            <SideNavItem>
              <SideNavLink href="/dashboard">Overview</SideNavLink>
            </SideNavItem>
            <SideNavItem>
              <SideNavLink href="/dashboard/analytics">Analytics</SideNavLink>
            </SideNavItem>
          </SideNavList>
        </Sidebar>

        {/* 2. 메인 콘텐츠 영역 */}
        <main className="flex-1 overflow-auto bg-[var(--bg-document)]">
          <header className="h-12 border-b flex items-center px-4">
             <SidebarToggle /> {/* 어디서든 토글 가능 */}
          </header>
          {children}
        </main>
      </div>
    </SidebarProvider>
  );
}
```
