# 페이지네이션 (Pagination)

많은 양의 데이터를 여러 페이지로 나누어 효율적으로 탐색할 수 있게 해주는 컴포넌트입니다.

<Callout type="note">
  사용자가 현재 위치를 쉽게 파악하고 원하는 페이지로 빠르게 이동할 수 있도록 직관적인 링크와 내비게이션 버튼을 제공합니다.
</Callout>

## 활용 가이드 (Usage)

### 필수 데이터
페이지네이션이 정상적으로 동작하려면 현재 페이지, 총 개수, 페이지당 항목 수 등의 정보가 필요합니다.

```tsx
<Pagination
  currentPage={currentPage}
  onPageChange={handlePageChange}
  totalCount={100}      // 전체 아이템 수
  pageViewRange={10}    // 한 페이지에 보여줄 아이템 수
  pageRange={5}         // 하단에 노출할 페이지 번호 개수
/>
```

---

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `currentPage` | `number` | `1` | 현재 활성화된 페이지 번호 |
| `totalCount` | `number` | `-` | 전체 아이템의 총 개수 |
| `pageViewRange` | `number` | `10` | 한 페이지당 노출할 아이템 수 |
| `pageRange` | `number` | `5` | 하단 내비게이션에 노출할 페이지 번호 최대 개수 |
| `onPageChange` | `(page: number) => void` | `-` | 페이지 변경 시 호출되는 콜백 함수 |
