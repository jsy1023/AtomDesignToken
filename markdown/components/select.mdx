# 선택창 (Select)

여러 개의 선택지 중 하나를 골라야 할 때 사용되는 드롭다운 형태의 컨트롤 요소입니다.

<Callout type="tip">
  선택지가 10개 이상으로 많아 화면에 모두 나열하기 어려울 때 사용을 권장합니다.
</Callout>

---
