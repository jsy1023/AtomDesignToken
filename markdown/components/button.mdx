# 버튼 (Button)

사용자가 클릭하여 액션을 수행할 수 있는 기본적인 인터랙티브 컴포넌트입니다.

<Callout type="tip">
  버튼 컴포넌트는 모든 표준 HTML 버튼 속성을 지원하며, 다양한 크기와 스타일 변형을 제공합니다.
</Callout>

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `type` | `primary` \| `secondary` \| `outline` \| `success` \| `warning` \| `danger` \| `gray` | `primary` | 버튼의 시각적 스타일 |
| `size` | `sm` \| `md` \| `lg` | `md` | 버튼의 크기 |
| `disabled` | `boolean` | `false` | 버튼 비활성화 여부 |
| `onClick` | `() => void` | - | 클릭 핸들러 |
