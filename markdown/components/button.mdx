# 버튼 (Button)

사용자가 클릭하여 액션을 수행할 수 있는 기본적인 인터랙티브 컴포넌트입니다.

<Callout type="tip">
  버튼 컴포넌트는 모든 표준 HTML 버튼 속성을 지원하며, 다양한 크기와 스타일 변형을 제공합니다.
</Callout>

---

가장 기본적인 버튼 사용 예시입니다. `name` 속성만으로 컴포넌트와 소스 코드를 자동으로 불러옵니다.

## 설치 (Installation)

<Steps>

### 패키지 추가
터미널에서 아래 명령어를 실행하여 버튼 컴포넌트를 프로젝트에 추가합니다.

## 속성 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `type` | `primary` \| `secondary` \| `outline` | `primary` | 버튼의 시각적 스타일 |
| `size` | `sm` \| `md` \| `lg` | `md` | 버튼의 크기 |
| `disabled` | `boolean` | `false` | 버튼 활성화 여부 |
| `loading` | `boolean` | `false` | 로딩 상태 표시 |
